KSS03186
Azionamenti scaricati (causa: {Causa della scarica})
Causa:I dati macchina non coincidono con il tipo di robot
Descrizione
Devono essere caricati i dati macchina corretti. La correttezza dei dati deve essere controllata confrontando i dati macchina caricati con i dati macchina sulla targhetta identificativa.
Quando si ricaricano i dati macchina, lo stato dei dati macchina deve corrispondere esattamente allo stato del KSS. Ciò è garantito quando ci si avvale dei dati macchina forniti insieme alla release KSS utilizzata.
Se vengono caricati i dati macchina sbagliati, il robot non deve essere messo in movimento! In caso contrario sussiste il pericolo di morte, gravi lesioni o significativi danni materiali. Devono essere caricati i dati macchina corretti.
Targhetta d'identificazione
Il percorso nel quale si trovano i dati macchina sul CD è indicato sulla targhetta identificativa nella riga ...\MADA\.
Per verificare se sono stati caricati i dati macchina corretti:
Presupposto
  • Modo operativo T1 o T2
  • Nessun programma selezionato.
Procedimento
  1. Nel menu principale selezionare Messa in servizio > Dati robot.
    Si apre la finestra Dati robot.
  2. Confrontare le seguenti informazioni:
    • Nella finestra Dati robot: indicazione nel campo Dati macchina
    • Sulla targhetta d'identificazione alla base del robot: indicazione nella riga $TRAFONAME()="# ..... "
Rimedio:Copia dati macchina
Descrizione
Con questa funzione è possibile copiare i dati macchina sul comando robot, ad esempio se è stato sostituito un robot.
Quando si caricano i dati macchina, lo stato dei dati macchina deve corrispondere esattamente allo stato del KUKA System Software (KSS). Ciò è garantito quando si caricano solo dati macchina forniti con la KSS-Release utilizzata (ad es. su una chiavetta USB).
Se vengono caricati i dati macchina sbagliati, il robot non deve essere messo in movimento! Sussiste altrimenti il rischio di lesioni fisiche e danni materiali.
Dopo modifiche ai dati macchina, è necessario adottare tutte le misure di sicurezza indicate nel manuale di utilizzo e di programmazione al capitolo “Sicurezza”. Sussiste altrimenti pericolo di morte, di gravi lesioni o di danni materiali.
Presupposto
  • Rilascio del System Software disponibile su drive locale o rete
  • Non è selezionato nessun programma.
  • Modo operativo T1 o T2
  • Gruppo d'utente Esperto
Procedimento
  1. Selezionare nel menu principale Messa in servizio > Coppia dati macchina. Viene visualizzata una struttura della directory.
  2. Vai alla Release del System Software.
  3. Nell’ambito della Release, vai ai dati macchine desiderati e seleziona la directory.
  4. Premi Copia. Vengono copiati i dati macchina. Si visualizza il messaggio Riconfigurazione in corso ….
    Il processo di copiatura si è concluso con successo se il messaggio non è più visualizzato o se oltre al messaggio si visualizza Copia dei dati macchina conclusa.
