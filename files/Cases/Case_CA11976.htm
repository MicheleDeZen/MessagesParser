KSS02256
Operando diverso da INT, REAL
Causa:Tipo di dati inammissibile per variabile loop
Descrizione
Per la variabile loop è stato utilizzato un tipo di dati come INT. Le variabili loop devono essere del tipo di dati "INT".
Sintassi
FOR Variabile loop = Valore iniziale TO Valore finale <STEP Incremento>
<Istruzioni>
ENDFOR
Spiegazione della sintassi
Elemento
Descrizione
Variabile loop
Tipo: INT
Variabile che conteggia gli svolgimenti dei programmi. È predefinito con il Valore iniziale. La variabile deve essere dichiarata in precedenza.
Il valore della variabile loop può essere utilizzato in istruzioni all’interno e all’esterno del loop. Dopo l’abbandono del loop, la variabile loop ha l’ultimo valore acquisito.
Valore iniziale; valore finale
Tipo: INT
Il Contatore deve essere occupato in via preliminare con il Valore iniziale. Dopo ogni ciclo, il Contatore si modifica automaticamente dell’incremento. Se si supera o non si raggiunge il Valore finale, il loop si conclude.
Incremento
Tipo: INT
Valore con il quale si modifica il Contatore a ogni ciclo. Il valore può essere negativo. Valore default: 1.
  • Valore positivo: Il loop si conclude se il contatore è superiore al valore finale.
  • Valore negativo: Il loop si conclude se il contatore è inferiore al valore finale.
Il valore non deve essere zero né una variabile.
Esempio
La variabile B aumenta di 1 in 5 svolgimenti dei programmi. La variabile loop ha denominazione A.
INT A
... 
FOR A=1 TO 10 STEP 2
   B=B+1 
ENDFOR
