KSS02187
Serie di caratteri attesa
Causa:Umlaut nel testo del messaggio
Descrizione
In un messaggio definito dall’utente è stato definito un testo di messaggio che contiene Umlaut.
Esempio di errata programmazione:
...
DECL KrlMsg_t mymessage
DECL KrlMsgPar_t mypar[3]
DECL KrlMsgOpt_t myoptions
DECL int handle
...
INI
...
mymessage = {modul[] "User", Nr 42, msg_txt[] "Würfel ablegen."}
handle = Set_KrlMsg (#notify, mymessage, mypar[], myoptions)
...
Esempio di corretta programmazione:
...
DECL KrlMsg_t mymessage
DECL KrlMsgPar_t mypar[3]
DECL KrlMsgOpt_t myoptions
DECL int handle
...
INI
...
mymessage = {modul[] "User", Nr 123, msg_txt[] "Wuerfel ablegen."}
handle = Set_KrlMsg (#notify, mymessage, mypar[], myoptions)
...
Per verificare se il messaggio definito dall’utente contiene Umlaut:
Istruzione verifica
  1. Aprire il programma.
  2. Cercare msg_txt[].
  3. Verificare se il testo dopo msg_txt[] contiene Umlaut.
Rimedio:Rimozione degli Umlaut dal testo del messaggio
Descrizione
Rimuovere gli Umlaut dal testo dopo msg_txt[] ad es. Wuerfel al posto di Würfel.
