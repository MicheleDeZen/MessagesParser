Cross3Archive 00069
Ripristino annullato! Mancano i seguenti pacchetti tecnologici nell'archivio ({0}) o nel controllo ({1})!
Causa:Archivio generato con versione Board Package inferiore a 1.3
Descrizione
Per la mainboard D3445-K è richiesta la versione 1.3 del Board Package. Dalla versione 1.3 il Board Package è trattato come pacchetto opzioni. Si è tentato di caricare un archivio generato da una versione precedente del Board Package.
Per verificare se il Board Package è elencato come pacchetto tecnologico:
Istruzione verifica
  1. Nel menu principale selezionare Messa in servizio > Software supplementare.
    Si apre la finestra Software addizionale installato.
  2. Verificare se il Board Package è riportato nell'elenco.
Rimedio:Convertire l'archivio
Descrizione
Gli archivi generati con una versione di Board Package precedente alla 1.3 devono essere convertiti con il programma KUKA.ArchiveAdapter. KUKA.ArchiveAdapter può essere scaricato qui www.kuka.com.
