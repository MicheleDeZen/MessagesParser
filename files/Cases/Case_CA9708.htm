KSS12032
scostamento di posizione (controllo del movimento <-> controllo di sicurezza)
Causa:Diversa configurazione tra dati macchine e comando di sicurezza
Descrizione
Il sistema ha registrato una differenza tra i dati macchina lato comando e i dati macchina depositati nella configurazione di sicurezza.
Possibile causa:
I dati macchina sono stati modificati tramite HMI nel file di configurazione $machine.dat. La modifica non viene trasferita automaticamente nella configurazione di sicurezza.
Rimedio:Aggiorna configurazione di sicurezza
Descrizione
Aprire la configurazione di sicurezza e acquisire la nuova configurazione.
Presupposto
  • Gruppo utenti Esperto o superiore
Se sul controllo robot è installata una delle opzioni KUKA.SafeOperation o KUKA.SafeRangeMonitoring, possono valere altri gruppi utenti. Per informazioni consultare le documentazioni delle opzioni menzionate.
Procedimento
  1. Selezionare nel menu principale Messa in servizio > Assistenza > Configurazione di sicurezza.
  2. La configurazione di sicurezza verifica se le modifiche siano rilevanti o meno per il comando di sicurezza.
    • Se le modifiche non sono rilevanti, si apre la finestra Configurazione di sicurezza. Non appare alcun messaggio. Non sono richieste azioni ulteriori.
    • Se le modifiche sono rilevanti, si visualizza un messaggio di dialogo. che segnala i dati macchina del controllo robot che differiscono da quelli del controllo di sicurezza.
      Il messaggio chiede se si intende aggiornare il controllo di sicurezza. Confermare l'interrogazione con Sì.
  3. Viene visualizzata un'interrogazione in merito all'applicazione o meno delle modifiche. Confermare l'interrogazione con Sì.
