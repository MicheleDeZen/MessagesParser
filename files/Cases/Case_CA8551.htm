KSS13039
Errore di inizializzazione del firmware Profinet
Causa:Conflitto di indirizzi IP nella rete con Profinet
Descrizione
All’interno della rete, due utenze hanno lo stesso indirizzo IP. Per questo motivo si verifica un conflitto di indirizzi IP.
Esempio
Un apparecchio collegato alla rete solo temporaneamente ha lo stesso indirizzo IP del robot.
Per verificare se è presente un conflitto di indirizzi IP:
Istruzione verifica
  1. Controllare l’indirizzo IP di ogni utenza di rete all’interno della rete, ad es. per gli apparecchi Windows tramite Esegui ... > cmd > ipconfig.
  2. Localizzare indirizzi IP doppi.
Rimedio:È necessario caricare di nuovo un avvio a freddo con file
Descrizione
Modificare l’indirizzo IP e riavviare il collegamento alla rete attraverso Caricare di nuovo avvio a freddo con file.
Presupposto
  • Gruppo d'utente Esperto
  • Privilegi di amministratore a livello di Windows
Procedimento
  1. Modificare l’indirizzo IP presente due volte nella rete per una delle utenze in questione.
  2. Nel menu principale selezionare Spegnimento.
  3. Nella finestra Spegnimento > Impostazioni standard per lo spegnimento con interruttore principale, selezionare il tipo Start a freddo.
  4. Nella finestra Spegnimento > Impostazioni valide solo allo spegnimento successivo con interruttore principale, selezionare Rinnova lettura file.
  5. Riavviare il PC di comando .
