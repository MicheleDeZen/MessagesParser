KSS14009
Boot non riuscito nel modulo: {Nome del modulo}
Causa:Disinstallazione incompleta di EtherNetIP
Descrizione
Durante la disinstallazione di EtherNetIP sul controllo nonché la cancellazione della struttura EtherNetIP in WorkVisual il file Modules_EIPIODriver.xml non viene disinstallato completamente.
File di configurazione
Directory
Directory di installazione WorkVisual\Config\User\Common\Modules_EIPIODriver.xml
File
Modules_EIPIODriver.xml
Per verificare se la disinstallazione è stata eseguita completamente:
Istruzione verifica
  1. Verificare se il parametro EIPIODriver è contenuto nella notifica.
  2. Aprire Windows Explorer.
  3. Navigare nella directory nella quale si dovrebbe trovare il file.
  4. Verificare se il file è presente nella directory.
Rimedio:Cancellazione di file da cartella e nuovo trasferimento del progetto
Descrizione
Cancellare il file interessato e trasferire di nuovo il progetto attualmente attivo sul controllo.
Presupposto
  • Gruppo d'utente Esperto
Procedimento
  1. Memorizzare il progetto attivo che attualmente si trova sul controllo.
  2. Navigare nella directory e cancellare il file interessato.
  3. Successivamente trasferire di nuovo il progetto salvato sul controllo.
  4. Nel menu principale selezionare Spegnimento.
  5. Nella finestra Spegnimento > Impostazioni standard per lo spegnimento con interruttore principale, selezionare il tipo Avvio a freddo.
  6. Nella finestra Spegnimento > Impostazioni valide solo allo spegnimento successivo con interruttore principale, selezionare Rinnova lettura file.
