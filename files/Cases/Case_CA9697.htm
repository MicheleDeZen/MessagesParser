KSS15127
Conf.: Arresto tramite sorveglianza zona di lavoro
Causa:Zona monitorata in T1 violata o superata
Descrizione
L’utensile di sicurezza attivo o una posizione dell'asse nel modo operativo T1 ha violato o superato una zona monitorata. Una segnalazione aggiuntiva indica inoltre quale zona è interessata.
Rimedio:Conferma del messaggio e spostamento del robot dalla zona violata
Descrizione
Il robot deve essere condotto nel modo operativo T1 fuori dalla zona violata. Ciò è possibile solo una volta confermato il messaggio.
Presupposto
  • Modo operativo T1
Procedimento
  1. Confermare il messaggio con OK.
  2. Premere il tasto di consenso e mantenerlo premuto.
  3. Spostare il robot nella direzione desiderata:
    • mediante i tasti di spostamento (modalità manuale)
    • mediante i tasti di Start e Start all'indietro (funzionamento del programma)
