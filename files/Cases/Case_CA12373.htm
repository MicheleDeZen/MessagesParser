MadaConfiguration 00007
Non esistono parametri servo Supply per {0} con un’intensità del resistore di {1}
Causa:KR CYBERTECH nano azionato con KR C4 compact per KR AGILUS
Descrizione
I KR C4 compact per le serie KR AGILUS e KR CYBERTECH nano dispongono delle stesse interfacce. Per motivi tecnici, per il funzionamento del KR CYBERTECH nano non è possibile utilizzare i controlli KR C4 compact che sono stati consegnati entro la FINE 2015. Per il funzionamento del KR CYBERTECH nano si devono usare solo KR C4 compact con la Drive Configuration DC4 (vedi targhetta).
Pertanto i robot della serie KR CYBERTECH nano non possono essere configurati con versioni software inferiori a KSS 8.3.18. Se quindi non si tenta, WorkVisual emette un messaggio di errore:
Per verificare quale versione software per il controllo è configurata in WorkVisual:
Istruzione verifica
  1. Carica progetto dal sistema di controllo.
  2. Verificare in Struttura del progetto sulla scheda di registro Dispositivi la voce della versione software dietro al controllo attivo.
Rimedio:Indicare in WorkVisual la versione KSS corretta
Descrizione
Impostare nel progetto WorkVisual per il controllo la versione KSS installata sul controllo.
Presupposto
  • Il progetto interessato è aperto.
Procedimento
  1. Sotto Struttura del progetto sulla scheda di registro Apparecchi fare clic con il tasto destro del mouse sul controllo interessato.
  2. Selezionare nel menu contestuale la voce Opzioni di controllo .
  3. Per la voce Versione firmware selezionare la versione KSS installata.
  4. Chiudere la finestra con OK.
