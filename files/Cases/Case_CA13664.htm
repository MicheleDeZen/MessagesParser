KSS03273
Spline troppo lunga
Causa:Carattere non riconosciuto o mancante
Descrizione
Il simbolo = è assente nel registro o non è stato riconosciuto.
In questo modo si può verificare se manca il carattere:
Presupposto
  • Gruppo d'utente Esperto
  • Privilegi di amministratore a livello di Windows
Istruzione verifica
  1. Passare al livello Windows.
  2. Premere il tasto di Windows Start e selezionare Esegui....
  3. Nel campo Apri inserire regedit e premere OK.
    Si apre la finestra Editor del Registro.
  4. Nella struttura ad albero selezionare la seguente cartella HKEY_CURRENT_USER\Software\VB e VBA Program Settings\KUKATPBASIS\CA_PRE.
  5. Verificare se in Default è presente il valore =.
Rimedio:Inserimento o sostituizione di simboli
Descrizione
Se nel registro manca il simbolo = occorre inserirlo. Se però è già presente il simbolo =, deve essere sostituito con un nuovo =.
Presupposto
  • Gruppo d'utente Esperto
  • Privilegi di amministratore a livello di Windows
Procedimento
  1. Passare al livello Windows.
  2. Premere il tasto di Windows Start e selezionare Esegui....
  3. Nel campo Apri inserire regedit e premere OK.
    Si apre la finestra Editor del Registro.
  4. Nella struttura ad albero selezionare la seguente cartella HKEY_CURRENT_USER\Software\VB e VBA Program Settings\KUKATPBASIS\CA_PRE.
  5. Selezionare Default e selezionare il tasto destro del mouse > Modify.
  6. Nel campo Value data inserire il simbolo = oppure sostituire il simbolo presente con un nuovo =.
  7. Fare clic su OK e passare al livello di Windows.
  8. Nel menu principale selezionare Spegnimento.
  9. Nella finestra Spegnimento > Impostazioni standard per lo spegnimento con interruttore principale, selezionare il tipo Avvio a freddo.
  10. Nella finestra Spegnimento > Impostazioni valide solo allo spegnimento successivo selezionare Ricarica dati.
