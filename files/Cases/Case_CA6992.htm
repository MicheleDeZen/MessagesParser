ConfigMon 00001
Gruppi non disponibili per l'utente corrente
Causa:Impossibile aprire la panoramica delle variabili perché ConfigMon.INI è stato rinominato o rimosso.
Descrizione
Il file ConfigMon.INI è presente di default sul comando robot. Comprende gruppi predefiniti. L’utente può elaborare i gruppi e crearne di nuovi.
Se il file ConfigMon.INI è stato rinominato o rimosso, il comando robot crea automaticamente un nuovo ConfigMon.INI se nel menù principale è stato selezionato uno dei seguenti comandi:
  • Visualizzazione > Variabile > In generale > Configurazione
  • Nel KUKA System Software:
    Visualizzazione > Variabile > In generale > Visualizza
  • Nel VW System Software:
    Visualizzazione > Variabile > In generale, pertanto uno dei gruppi predefiniti
Il ConfigMon.INI auto-generato non contiene gruppi. Questo lo distingue dal Default-ConfigMon.INI.
Se è disponibile un ConfigMon.INI auto-generato, ciò significa che il file originale è stato rinominato o rimosso.
Per verificare se è presente un ConfigMon.INI auto-generato:
Presupposto
  • Gruppo d'utente Esperto
Istruzione verifica
  1. Nel menu principale selezionare Visualizzazione > Variabile > In generale > Edit "ConfigMon.ini".
  2. Se ConfigMon.INI sembra uguale a quanto rappresentato nella seguente figura, esso è stato generato automaticamente dal comando robot. Ciò significa che il file originale è stato rinominato o rimosso.
ConfigMon.INI auto-generato
Rimedio:Ripristino o riconfigurazione di ConfigMon.INI
Presupposto
  • Gruppo d'utente Esperto
Procedimento
  1. Nel navigatore, all’interno della directory C:\KRC\USER, verificare se l’originario (eventualmente rinominato per errore) ConfigMon.INI è ancora identificabile.
  2. In caso positivo:
    Nel navigatore, all’interno della directory C:\KRC\USER cancellare il ConfigMon.INI auto-generato e rinominare il file originale in "ConfigMon.INI".
    Il comando robot non deve essere riavviato. Il passaggio 3 e gli altri passaggi di questa procedura non devono essere ripetuti.
  3. In caso negativo:
    Se non sono disponibili copie di sicurezza di ConfigMon.INI, da aggiungere in C:\KRC\USER, è necessario riconfigurare ConfigMon.INI.
    A tale scopo, nel menu principale selezionare Visualizzazione > Variabile > In generale > Edit "ConfigMon.ini".
  4. Dopo la riga con la versione, digitare le righe [Group1] e altre, come raffigurato sotto.
    Il file contiene adesso un gruppo esemplare con una variabile.
  5. Chiudere il file ConfigMon.INI. Alla richiesta di conferma, se si intendono salvare le modifiche, rispondere con Sì.
  6. Nel menu principale selezionare Visualizzazione > Variabile > In generale > Configurazione.
    Si apre la finestra Visualizzazione variabili - configurazione.
    Ciò è possibile solo perché ConfigMon.INI contiene già un gruppo. Senza gruppo, la panoramica delle variabili non può essere aperta.
  7. Secondo necessità, elaborare il gruppo esistente e aggiungerne altri.
    Per ulteriori informazioni sulla configurazione della panoramica variabili consultare le istruzioni operative e di programmazione per gli integratori di sistema.
  8. Premere Ok per salvare la configurazione e chiudere la finestra.
