KSS01422
{Variabile $} valore non valido
Causa:Il segnale non è associato ad alcun ingresso né ad alcuna uscita
Descrizione
Per utilizzare una variabile dichiarata come segnale, è necessario prima associarla a un ingresso o a un’uscita. I segnali di sistema KUKA inutilizzati sono preimpostati da standard con FALSE. Questi segnali non possono essere utilizzati.
Esempio di associazione corretta di un’uscita:
SIGNAL START_PROCESS $OUT[7]
Sintassi
Accordo di nomi segnali per ingressi o uscite:
SIGNAL Nome segnale variabile segnale <TO Variabile segnale>
Chiusura di un accordo SEGNALE predefinito nel sistema:
SIGNAL Nome segnale sistema FALSE
Spiegazione della sintassi
Elemento
Descrizione
Nome segnale
Nome a piacere
Variabile di segnale
Variabili di segnale predefinite. È possibile selezionare i seguenti tipi:
  • $IN[x]
  • $OUT[x]
  • $ANIN[x]
  • $ANOUT[x]
TO
Riassume diversi ingressi o uscite binari in sequenza (max. 32) in un ingresso o uscita digitale. I segnali riassunti possono essere affrontati a livello decimale, esadecimale (prefisso H) o come bitmuster (prefisso B). È possibile anche un’elaborazione con operatori booleani.
Nome segnale di sistema
Il nome di segnale predefinito nel sistema, ad es. $T1.
FALSE
Chiude un accordo SEGNALE predefinito nel sistema. Gli ingressi o le uscite che si riferiscono all’accordo SEGNALE sono pertanto di nuovo disponibili per altri scopi.
FALSE non è qui un valore booleano ma una parola chiave. L’opzione TRUE non è disponibile. Se l’accordo SEGNALE chiuso con FALSE deve essere di nuovo valido, la riga di programma contenente FALSE deve essere cancellata.
Istruzione verifica
  • Controllare se l’accordo di segnale in questione è associato a un ingresso o a un’uscita.
Rimedio:Modifica di variabile o di parametro
Presupposto
  • Gruppo d'utente Esperto
A seconda della configurazione e del software di sistema utilizzato, possono essere necessarie autorizzazioni utente superiori.
Procedimento
  1. Selezionare il file nel navigatore e premere Apri. Il file viene visualizzato nell'editor.
  2. Cercare il punto corrispondente nel file e modificare il valore.
  3. Salvare il file e chiuderlo.
