KSS02098
Variabile non è dichiarata nella lista DAT
Causa:Variabile nella lista dati non dichiarata
Descrizione
La variabile non è stata dichiarata nella lista dati. La lista dati contiene dati e coordinate dei punti permanenti. La lista dati è anche detta File DAT.
Così si può verificare se le variabili della lista dati sono state dichiarate:
Presupposto
  • Gruppo d'utente Esperto
Istruzione verifica
  1. Aprire il file DAT appartenente al file SRC attualmente in esecuzione.
  2. Selezionare nella barra menu Elabora > Cerca.
  3. Controllare se il nome della variabile emerge nella lista dati.
Rimedio:Dichiarare la variabile nell'elenco dati
Descrizione
Dichiarare la variabile nell'elenco dati.
Una variabile è dichiarata nell’elenco dati nel seguente modo:
Sintassi
Dichiarazione di variabili negli elenchi dati:
<DECL> <GLOBAL> Tipo di dato Nome1 <, ..., NomeN>
Dichiarazione di variabili negli elenchi dati, con contemporanea inizializzazione:
<DECL> <GLOBAL> Tipo di dato Nome = Valore
Alla dichiarazione con contemporanea inizializzazione, per ogni variabile è necessario un accordo DECL proprio. Non è possibile dichiarare e inizializzare più variabili con un unico accordo DECL.
Spiegazione della sintassi
Elemento
Descrizione
DECL
Si può tralasciare il DECL, se il Tipo di dati è predefinito. Se il Tipo di dati è definito dall’utente, il DECL è obbligatorio.
GLOBAL
Ambito di validità della variabile
Tipo di dati
Indicazione del tipo di dati desiderato
Nome
Nome dell’oggetto (variabile, campo o costante) che viene dichiarato
Valore
Il tipo di dato di Valore deve essere compatibile con il Tipo di dato, ma non identico. In caso di tipi di dati compatibili, il sistema svolge automaticamente un adattamento del tipo.
Procedimento
  1. Aprire il file DAT nel navigatore.
  2. Dichiarare la variabile.
  3. Salvare il file DAT e chiuderlo.
