KSS02144
Manca ENDLOOP
Causa:Il loop non è stato concluso
Descrizione
Attraverso il programma LOOP è possibile programmare in un programma un loop. Il loop deve essere concluso mediante il comando ENDLOOP.
Esempio di errata programmazione:
 
DEF Prog()
LOOP
...
END
 
Esempio di corretta programmazione:
 
DEF Prog()
LOOP
...
ENDLOOP
END
 
Per verificare se dopo un LOOP è stato programmato un ENDOOP:
Istruzione verifica
  1. Selezionare il programma e fare clic su Apri.
  2. Cercare per LOOP e verificare se il relativo comando ENDLOOP è stato programmato.
Rimedio:Modificare il programma.
Presupposto
  • Gruppo utenti Esperto
  • Modo operativo T1, T2 o AUT
Procedimento
  1. Selezionare il programma nel navigatore e premere Apri. Il programma viene visualizzato nell'editor.
  2. Cercare ed elaborare il punto corrispondente nel programma.
  3. Chiudere il file e alla richiesta di conferma, se si intende salvare le modifiche, rispondere con Sì.
