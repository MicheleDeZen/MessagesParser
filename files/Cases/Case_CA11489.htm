KSS01009
Conversione punto senza modello accuratezza assoluta non possibile.
Causa:Conversione in posizioni Robot cartesiane precise possibile solo con robot con precisione assoluta
Descrizione
La variabile $ABS_CONVERT serve a convertire le posizioni del robot apprese senza modello robot con posizionamento preciso, in posizioni robot precise. Se la funzione è attiva, pertanto la variabile ha il valore TRUE, anche il modello robot con precisione assoluta deve essere attivo.
Questo modello robot con precisione assoluta è attivato tramite la variabile $ABS_ACCUR.
Fisicamente, le posizioni robot convertite sono eseguite senza modifica. Se per un punto sono depositate coordinate di posizionamento preciso, Bit 5 mostra l’indicazione di Stato. Se si applica il Bit (Bit 5 = 1), si tratta di coordinate di posizionamento preciso e il punto non è convertito nuovamente.
Sintassi
$ABS_CONVERT = state
Spiegazione della sintassi
Elemento
Descrizione
state
Tipo: BOOL
  • TRUE: Conversione in posizioni robot cartesiane con posizionamento preciso
  • FALSE: Nessuna conversione in posizioni robot cartesiane con posizionamento preciso
Default: FALSE
In questo modo si può verificare se la variabile di sistema $ABS_ACCUR è su TRUE:
Istruzione verifica
  1. Selezionare nel menu principale Visualizzazione > Variabile > Singolo.
    Si apre la finestra Visualizzazione della variabile – singola.
  2. Nel campo Nome immettere il nome della variabile e confermare con il tasto Enter.
    Nel campo Valore attuale viene visualizzato il valore attivo delle variabili.
Rimedio:Modifica di variabile o di parametro
Presupposto
  • Gruppo d'utente Esperto
A seconda della configurazione e del software di sistema utilizzato, possono essere necessarie autorizzazioni utente superiori.
Procedimento
  1. Selezionare il file nel navigatore e premere Apri. Il file viene visualizzato nell'editor.
  2. Cercare il punto corrispondente nel file e modificare il valore.
  3. Salvare il file e chiuderlo.
