Navigator 00030
Ripristino non riuscito
Causa:Protezione dalla scrittura su file o file .SRC e .DAT copiati manualmente
Descrizione
Se nella directory di destinazione sono presenti file protetti dalla scrittura, non è possibile ripristinare l'archivio.
Non è nemmeno possibile ripristinare i file protetti da scrittura.
Questo vale per tutti i tipi di file, ad esclusione di .SRC e .DAT, che vengono sempre sovrascritti automaticamente. Se i file .SRC e .DAT vengono copiati manualmente nel sistema di file e non all'interno del navigatore, verranno trattati come gli altri file e impediranno il ripristino.
Rimedio:Togliere l’attributo protezione da scrittura.
Descrizione
È necessario rimuovere la protezione da scrittura per il file.
Presupposto
  • Gruppo utenti Administrator
Procedimento
  1. Aprire l'Explorer.
  2. Aprire la cartella in cui si trova il file.
  3. Aprire le proprietà del file attraverso la sequenza di menu File > Proprietà.
  4. Nella scheda di registro Generale disattivare l’Attributo Protezione da scrittura.
  5. Confermare con OK.
