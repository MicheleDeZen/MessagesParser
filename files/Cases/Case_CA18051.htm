KSS26112
Avvertenza: temperatura del raffreddatore eccessiva ({Tipo di apparecchio}) ({Numero}).
Causa:Distanze minime non osservate
Descrizione
La temperatura del corpo di raffreddamento è eccessiva perché non sono state rispettate le distanze minime.
Istruzione verifica
Verificare le distanze minime del controllo robot.
Nella figura sono rappresentate le distanze minime da rispettare per il controllo robot.
Distanze minime
Danni alle cose in caso di mancato mantenimento delle distanze minime
Se non vengono rispettate le distanze minime c'è pericolo di danneggiare il controllo robot.
  • Rispettare rigorosamente le distanze minime indicate.
Determinati lavori di manutenzione e riparazione sul controllo robot devono essere eseguiti sul lato o sul retro del controllo. Per tale motivo il controllo robot deve risultare bene accessibile. In caso i pannelli laterali o quello posteriore non siano accessibili, deve essere possibile spostare il controllo robot in una posizione che permetta di eseguire i lavori.
Rimedio:Osservare le distanze minime
Procedimento
  • Osservare le distanze minime
