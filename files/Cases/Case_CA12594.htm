KSS27009
Efficacia freno{Freno n.}{Asse n.} OK
Causa:File log del test dei freni troppo grande
Descrizione
Il file log del test dei freni è troppo grande e pertanto i messaggi vengono emessi ciclicamente.
File log
Directory
C:\KRC\Roboter\LOG\
File
BrakeTest.log
Per verificare se il comportamento è stato attivato da un file log troppo grande:
Procedimento
  1. Verificare se i messaggi scompaiono se viene eseguita una delle seguenti azioni:
    • Cambio del modo operativo
    • Deselezione del programma
    • Riavvio del controllo
  2. Verificare se il file log è grande diversi megabyte (circa 11 MB).
Rimedio:Cancellazione di file
Descrizione
Cancellare il file in questione dalla directory.
Si consiglia di archiviare il file prima della cancellazione.
Requisito
  • Livello Windows
Procedimento
  1. Navigare nella directory nella quale si trova il file.
  2. Selezionare il file.
  3. Cancellazione del file.
