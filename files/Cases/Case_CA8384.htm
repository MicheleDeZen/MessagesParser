KSS15081
Superato campo di sorveglianza n. {Numero del campo di sorveglianza}
Causa:Zona monitorata in T1 o T2 superata
Descrizione
Il vano di monitoraggio è stato superato in modalità T1 o T2. L'utensile di sicurezza attivo o perlomeno uno degli assi robot non si trova più nella zona consentita della zona monitorata.
Se il superamento dello spazio si verifica nel modo operativo T1, viene visualizzata anche la segnalazione di conferma Conf.: Arresto tramite sorveglianza zona di lavoro.
Si parla di superamento della zona monitorata quando il monitoraggio era già attivo quando si è superato il limite di spazio.
La zona consentita dipende dal tipo di zona monitorata:
Tipo spazio
Spazio di lavoro
Zona protetta
Spazio cartesiano
L'utensile di sicurezza attivo deve spostarsi entro i confini della zona monitorata.
Superamento dello spazio quando l'utensile di sicurezza abbandona la zona monitorata.
L'utensile di sicurezza attivo deve spostarsi al di fuori dei confini della zona monitorata.
Superamento dello spazio quando l'utensile di sicurezza si sposta nella zona monitorata.
Zona assi
Gli assi devono sempre muoversi entro i limiti della zona monitorata.
Superamento dello spazio quando gli assi abbandonano la zona monitorata.
Gli assi devono sempre muoversi al di fuori dei limiti della zona monitorata.
Superamento dello spazio se gli assi si muovono nella zona monitorata.
Rimedio:Condurre il robot fuori dalla zona violata
Descrizione
Il robot deve essere condotto nel modo operativo T1 fuori dalla zona violata. Non è impostabile un’altra modalità fino a che il robot non ha abbandonato la zona violata.
Presupposto
  • Modo operativo T1
Procedimento
  1. Se viene visualizzato un messaggio di conferma, confermarlo con OK.
  2. Premere il pulsante di consenso e mantenerlo premuto.
  3. Condurre il robot fuori dalla zona violata:
    • mediante i tasti di spostamento (modalità manuale)
    • mediante il tasti di Start e Start all'indietro (funzionamento del programma)
