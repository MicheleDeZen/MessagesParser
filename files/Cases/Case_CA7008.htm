KSS10033
{Nome istanza} Errore di connessione al firmware
Causa:Nessuna comunicazione con il firmware
Descrizione
Impossibile stabilire una connessione con la scheda del bus di campo.
Rimedio:Attivare il riavvio automatico in IBSPCI.XML
Descrizione
Per attivare il riavvio automatico, è necessario settare la voce RESET in ISBSPCI.XML su 1.
Nota
Valore
Funzione
RESET
0 ... 1
Default: 0
Per portare l’accoppiatore di sistema all’avvio in uno stato definito, prima dell’inizializzazione è necessario effettuare un reset.
0: All’avvio del driver non si svolge alcun reset.
1: All’avvio si svolge un reset e si aspetta per max. 7 s che l’accoppiatore di sistema sia di nuovo pronto per l’utilizzo. Se l’accoppiatore di sistema non può essere sollecitato entro 7 s, il caricamento del driver si interrompe.
Presupposto
  • Livello Windows (CTRL+ESC)
Procedimento
  1. Aprire il file C:\KRC\ROBOTER\Config\User\Common\IBSPCI.XML.
  2. Controllare le impostazioni applicate nei singoli segmenti e se necessario adattarle.
  3. Salvare le modifiche e chiudere il file.
Le modifiche devono sempre essere acquisite nel menu principale Configurazione > Inputs/outputs > Driver I/O.
