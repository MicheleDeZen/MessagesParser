KSS10006
{Nome istanza} Errore apertura file {Nome file}
Causa:Il file di configurazione SVC non è presente
Descrizione
Il file di configurazione SVC deve essere nella directory C:\KRC\Roboter\Config\User\Common.
In questo modo si può verificare se il file è presente.
Presupposto
  • Il navigatore è visualizzato.
  • Gruppo d'utente Esperto
Istruzione verifica
  1. Verificare se il parametro Nome file nella notifica {Nome istanza} Errore apertura file {Nome file} fa riferimento a un file SVC.
  2. Nel navigatore, aprire la directory C:\KRC\Roboter\Config\User\Common.
  3. Verificare se il file di configurazione SVC è disponibile.
Rimedio:Salvare il file di configurazione SVC tramite chiavetta USB nella directory
Descrizione
Il file di configurazione SVC deve essere generato con il tool Interbus (Config+) e poi, tramite la chiavetta KUKA.USBData, deve essere depositato nella directory C:\KRC\Roboter\Config\User\Common.
Presupposto
  • Il navigatore è visualizzato.
  • Gruppo d'utente Esperto
  • È collegata una chiavetta KUKA.USBData con il file di configurazione SVC.
Procedimento
  1. Nel navigatore, aprire la chiavetta KUKA.USBData e contrassegnare il file di configurazione SVC.
  2. Premere il pulsante Edit e selezionare il comando Copia.
  3. Nel navigatore, aprire la directory C:\KRC\Roboter\Config\User\Common.
  4. Premere il pulsante Edit e selezionare il comando Inserisci.
    Il file di configurazione SVC è copiato nell’elenco.
  5. Svolgere un avvio a freddo del comando.
