KSS27014
Dati configurazione non validi per il freno{Freno}{Asse} (codice errore {Codice di errore})
Causa:Limitazione della corrente per l’asse attiva
Descrizione
Per l’asse in questione è attiva una limitazione della corrente. La limitazione serve per proteggere il riduttore e non deve essere rimossa. In questo caso la limitazione della corrente impedisce l'esecuzione del test freni.
Per verificare se è attiva una limitazione della corrente per quell’asse:
Presupposto
  • Gruppo d'utente Esperto
File di configurazione
Directory
C:\KRC\Roboter\config\common\user\mada\NGaxis\
File
EAchsindex.xml
Istruzione verifica
  1. Aprire il file di configurazione dell’asse interessato sul controllo oppure aprire in WorkVisual.
  2. Ricerca per elemento <Limit>.
  3. Se l’elemento Limit contiene l’attributo Current con il corrispondente valore, è attiva una limitazione della corrente.
    Esempio: <Limit Speed="0" Current="8" />
    Limitazione corrente 8 A attiva
Rimedio:Rimuovere l’asse dal test freni
Descrizione
Per poter eseguire correttamente il test freni per gli assi del robot, occorre rimuovere l'asse in questione dal test freni.
Presupposto
  • Il progetto è aperto in WorkVisual.
  • Il sistema di controllo del robot è impostato su attivo.
Procedimento
  1. Nella finestra Struttura del progetto nella scheda di registro Apparecchi fare doppio clic sul corrispondente asse esterno da modificare. Si apre l'editor.
  2. Togliere il segno di spunta sul parametro Attivare test di freni.
  3. Memorizzare il progetto per applicare le modifiche.
