KSS02038
Dichiarazione non è nella sezione di dichiarazione
Causa:Dichiarazione programmata fuori dalla sezione di dichiarazione
Descrizione
La dichiarazione è stata programmata fuori dalla sezione di dichiarazione.
Per verificare se la dichiarazione è stata eseguita nella sezione di dichiarazione:
Presupposto
  • Gruppo utente "Esperto"
  • Il programma è stato selezionato o aperto.
Istruzione verifica
  1. Selezionare la sequenza di menu Edit > Vista.
  2. Attivare la voce di menu Riga DEF.
    La riga DEF è visualizzata.
  3. Verificare se la dichiarazione è stata programmata nella sezione di dichiarazione.
    DEF main()
    ; Deklarationsteil
    ; Hier muss die Deklaration vorgenommen werden.
    ...
    ; Initialisierungsteil
    INI
    ...
    ; Anweisungsteil
    SPTP HOME Vel=100% DEFAULT
    ...
    END
Rimedio:Correggere il programma
Procedimento
  • Correggere il programma
    Per maggiori informazioni, consultare la documentazione del rispettivo software.
