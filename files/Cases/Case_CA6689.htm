KSS10020
{Nome istanza} Errore di accesso al gruppo di collegamento
Causa:Nessuna comunicazione con il gruppo di accensione
Descrizione
Il numero di board in IBSPCI.XML non coincide con il numero di board impostato sul gruppo di accensione.
Rimedio:Adattare il numero di board in IBSPCI.XML
Descrizione
Il gruppo di accensione si compone o di una scheda Master e Slave o solo di una scheda Master. A tale scheda in IBSPCI.XML deve essere associato un numero di Board che coincide con il numero di Board impostato sul gruppo di accensione.
Nota
Valore
Funzione
BOARDNUMBER
1 ... 8
Default: 1
Assegna all’accoppiatore di sistema una sigla univoca per il canale dati. Tale sigla deve coincidere con il numero di ordine selezionato tramite il tasto DIP sul gruppo master.
Presupposto
  • Livello Windows (CTRL+ESC)
Procedimento
  1. Aprire il file C:\KRC\ROBOTER\Config\User\Common\IBSPCI.XML.
  2. Controllare le impostazioni applicate nei singoli segmenti e se necessario adattarle.
  3. Salvare le modifiche e chiudere il file.
Le modifiche devono sempre essere acquisite nel menu principale Configurazione > Inputs/outputs > Driver I/O.
