KSS02352
$OUT_C(n) non disponibile qui
Causa:$OUT_C[n] viene utilizzato su un punto non consentito
Descrizione
Con la variabile è possibile definire o resettare una uscita digitale con riferimento alla corsa principale. Ad esempio, l’utente può utilizzare la variabile per definire una uscita digitale nel momento di un movimento con arresto preciso o al vertice di un movimento di approssimazione.
La variabile non può essere interrotta in un trigger o interrupt.
Sintassi
$OUT_C[output number] = value
Spiegazione della sintassi
Elemento
Descrizione
Numero uscita
Tipo: INT
  • 1 … 4 096: KUKA System Software 8.1
  • 1 … 8 192: KUKA System Software 8.2 o superiore
Nota: il rispettivo campo di valori contiene il numero massimo possibile di I/O digitali. Il numero di I/O digitali effettivamente disponibili può essere interrogato tramite $NUM_IN/$NUM_OUT.
Stato
Tipo: BOOL
  • TRUE o FALSE
Per verificare se la variabile viene utilizzata in un trigger o interrupt:
Istruzione verifica
  1. Selezionare il programma e fare clic su Apri.
  2. Cercare trigger o interrupt e verificare se nel comando la variabile $OUT_C[] è stata programmata.
Rimedio:Modificare il programma.
Presupposto
  • Gruppo utenti Esperto
  • Modo operativo T1, T2 o AUT
Procedimento
  1. Selezionare il programma nel navigatore e premere Apri. Il programma viene visualizzato nell'editor.
  2. Cercare ed elaborare il punto corrispondente nel programma.
  3. Chiudere il file e alla richiesta di conferma, se si intende salvare le modifiche, rispondere con Sì.
