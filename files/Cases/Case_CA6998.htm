KSS10006
{Nome istanza} Errore apertura file {Nome file}
Causa:La configurazione del file SVC è errata
Descrizione
Attraverso una configurazione errata del file SVC non è possibile avviare il driver interbus.
In questo modo non è possibile verificare la configurazione del file.
Presupposti
  • L’Interbus Tool (Config+) per la creazione e la verifica del file di configurazione SVC è presente.
Istruzione verifica
  1. Verificare se il parametro Nome file nella notifica {Nome istanza} Errore apertura file {Nome file} fa riferimento a un file SVC.
  2. Verificare la configurazione del file SVC mediante l’Interbus Tool (Config+).
Per ulteriori informazioni sulla configurazione del file SVC, consultare la documentazione del tool Interbus (Config+).
Rimedio:Salvare il file di configurazione SVC tramite Config+ nella directory
Descrizione
Il file di configurazione SVC deve essere generato con il tool Interbus (Config+) e depositato nella directory C:\KRC\Roboter\Config\User\Common.
Il file è copiato automaticamente, dopo la generazione con il tool Interbus (Config+), nella directory C:\KRC\Roboter\Config\User\Common.
Presupposto
  • Il tool Interbus (Config+) è disponibile.
Procedimento
  1. Generare il file di configurazione SVC tramite un tool Interbus (Config+).
    Il file è copiato automaticamente, dopo la generazione con il tool Interbus (Config+), nella directory C:\KRC\Roboter\Config\User\Common.
    Per ulteriori informazioni sulla configurazione del file SVC, consultare la documentazione del tool Interbus (Config+).
  2. Svolgere un avvio a freddo del comando.
