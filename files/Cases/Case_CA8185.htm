KSS15053
Conf.: Gruppo di riferimento calibrazione n. {Numero del gruppo di riferimento} non referenziato
Causa:Gruppo di riferimento non acquisito
Descrizione
Gruppo di riferimento non acquisito.
Per il gruppo di riferimento devono essere acquisiti i seguenti punti:
  • Movimento verso il punto di riferimento
  • Posizione di riferimento
    La posizione di riferimento deve essere appresa nella configurazione di sicurezza.
  • Movimento via dal tasto di riferimento
I punti sono acquisiti nel seguente file:
File
Directory
C:\KRC\Roboter\KRC\R1\Program
File
masref_user.src
Il file contiene 2 subroutines per gruppo di riferimento 1 … 3.
MASREFSTARTG1() … MASREFSTARTG3()
Qui si acquisisce il movimento verso il punto di riferimento e la posizione di riferimento.
MASREFBACKG1() … MASREFBACKG3()
Qui si acquisisce il movimento dal punto di riferimento.
Istruzione verifica
  1. Selezionare il file nel navigatore e premere Apri. Il file viene visualizzato nell'editor.
  2. Controllare se i punti necessari per avviare un gruppo di riferimento sono stati acquisiti.
Rimedio:Acquisizione gruppo di riferimento
Descrizione
I punti necessari per avviare un gruppo di riferimento devono essere acquisiti.
Quest'attività va eseguita secondo il procedimento descritto nelle istruzioni per l'uso e per il montaggio.
