KSS01516
Sistema di riferimento non programmato
Causa:La base non è calibrata o è calibrata in modo errato
Descrizione
La base è stata calibrata in modo errato o non è stata proprio calibrata.
Presupposto
  • Gruppo d'utente Esperto
Per verificare se la base è stata calibrata:
Istruzione verifica
  1. Selezionare nel menu principale Messa in servizio > Misurazione > Punti di misura e quindi la voce di menu desiderata:
    • Tipo utensile
    • Tipo base
    • Asse esterno
  2. Immettere il numero dell'utensile, della base o della cinematica esterna.
    Vengono visualizzati il metodo di misurazione e i dati di misurazione.
Rimedio:Calibrazione della base
Descrizione
La base deve essere calibrata di nuovo.
Informazioni relative al metodo di calibrazione della base sono riportate nelle istruzioni per l’uso del KUKA System Software.
