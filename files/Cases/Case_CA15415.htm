KSS27007
Momento di mantenimento freno{Freno n.}{Asse n.} insufficiente
Causa:Valori inadatti per il test freni su un’unità lineare
Descrizione
I valori predefiniti per il test freni nel file mdrBrakeTest.ini sono impostati su assi rotatori. Per il test freni su un asse lineare, è necessario un adattamento.
La variabile TravAngle... definisce fino a che punto debbano essere movimentati gli assi per il test freni. In caso di assi rotatori, secondo l’impostazione predefinita sono 15°. In caso di assi lineari, il valore deve essere aumentato (ad es. 100 mm).
Per verificare su quale valore sia impostata la variabile TravAngleE1:
File di configurazione
Directory
C:\KRC\ROBOTER\Config\User\Common\Mada\MotionDrivers
File
mdrBrakeTest.ini
Presupposto
  • Gruppo d'utente Esperto
Istruzione verifica
  1. Selezionare il file nel navigatore e premere Apri. Il file viene visualizzato nell'editor.
  2. Cercare la variabile o il parametro e verificare il valore.
Ulteriori informazioni:
  • Consiglio
Rimedio:Modifica di variabile o di parametro
Presupposto
  • Gruppo d'utente Esperto
A seconda della configurazione e del software di sistema utilizzato, possono essere necessarie autorizzazioni utente superiori.
Procedimento
  1. Selezionare il file nel navigatore e premere Apri. Il file viene visualizzato nell'editor.
  2. Cercare il punto corrispondente nel file e modificare il valore.
  3. Salvare il file e chiuderlo.
