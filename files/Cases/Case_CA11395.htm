KSS00201
Pulsante uomo morto rilasciato
Causa:Pulsante di consenso sullo smartPAD difettoso
Descrizione
Il pulsante di consenso sullo smartPAD è difettoso.
Tasto di consenso
Per verificare se il pulsante di consenso sullo smartPAD è difettoso:
Istruzione verifica
  1. Azionare il tasto di consenso.
  2. Controllare lo stato delle variabili di sistema $SAFETY_SW.
Sintassi
$SAFETY_SW=Stato
Spiegazione della sintassi
Elemento
Descrizione
Stato
Tipo: ENUM
  • #PRESSED: Un pulsante di consenso è premuto (posizione centrale).
  • #RELEASED: Nessun pulsante di consenso premuto oppure pulsante di consenso premuto (posizione di panico).
Rimedio:Sostituzione dispositivo di comando
Descrizione
Alla sostituzione del dispositivo di comando assicurarsi che solo il giusto dispositivo di comando possa essere utilizzato.
