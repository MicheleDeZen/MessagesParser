KSS03009
Proprietà di {Denominazione area di lavoro} persa - (cold booted controller)
Causa:Gamma di lavoro non più bloccata attraverso il comando locale
Descrizione
La gamma di lavoro non è più bloccata attraverso il comando locale e il robot potrebbe non trovarsi nello spazio di lavoro.
Questo stato può ad esempio intervenire in caso di avvio a freddo del comando.
Per verificare lo stato dello spazio di lavoro:
Istruzione verifica
  1. Selezionare nel menu principale Visualizza > Spazio di lavoro.
    Nella finestra Stato spazio di lavoro si visualizzano gli stati dei robot.
    Simbolo
    Descrizione
    Il robot si trova nello spazio di lavoro.
    Il robot attende l’autorizzazione dello spazio di lavoro.
    Auto-blocco ("Deadlock") dello spazio di lavoro.
    Stato sconosciuto, possibili problemi di collegamento.
  2. Con Precedente / Successivo cambiare spazi di lavoro.
Rimedio:Ripristinare lo stato degli spazi di lavoro
Descrizione
Con la variabile $WORKSPACERESTOREACTIVE si può ripristinare lo stato delle aree di lavoro.
Se la variabile è impostata su TRUE, dopo un avvio a freddo si ripristina lo stato delle aree di lavoro sul comando robot.
Poi, il comando controlla se il file C:\KRC\Roboter\Init\WSRestore.INI esiste. Questo file viene creato automaticamente allo spegnimento del comando. Con questo file, il comando ripristina lo stato delle aree di lavoro.
Se il file WSRestore.INI non esiste, si interrogano le aree di lavoro dei comandi coinvolti. Da questi dati è possibile una ricostruzione delle aree di lavoro sul comando attuale.
La variabile è impostata di default su FALSE.
Una ricostruzione completa interrogando i comandi coinvolti non può essere garantita. Lo stato di ogni area di lavoro deve pertanto essere controllato ed eventualmente corretto.
Presupposto
  • Gruppo d'utente Esperto
Procedimento
  1. Nel menu principale selezionare Visualizzazione > Variabile > Singolo.
    Si apre la finestra Visualizzazione della variabile – singola.
  2. Nel campo Nome immettere il nome della variabile e confermare con il tasto Enter. Viene visualizzato il valore attuale delle variabili.
  3. Inserire il valore nuovo nel campo Valore nuovo.
  4. Premere il pulsante Setta valore. Nel campo Valore attuale viene visualizzato il nuovo valore.
