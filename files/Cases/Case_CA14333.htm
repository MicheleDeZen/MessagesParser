TVW00110
Macro di inizializzazione macro{MakroNumberSpecialRun} non disponibile. Inizializzazione impossibile.
Causa:Non è stata creata ancora alcuna macro di inizializzazione
Descrizione
La macro di inizializzazione viene richiamata automaticamente nei seguenti casi:
  • In caso di start a freddo
  • In caso di riavvio dell’interprete Submit del sistema
La macro di inizializzazione non è presente di default nel controllo robot. Deve essere creata dall'utente ed eventualmente configurata.
MakroNumberSpecialRun
La variabile MakroNumberSpecialRun definisce quale, macro verrà chiamata dal controllo come macro di inizializzazione.
Default: 57
Rimedio:Creazione e configurazione della macro di inizializzazione
Presupposto
  • Gruppo utenti Utente
  • Il navigatore è visualizzato.
Procedimento
  1. Creazione di una macro di inizializzazione
    1. Nella struttura di directory selezionare la cartella Macro.
    2. Premere il pulsante Nuovo.
    3. Solo nel gruppo utenti Esperto:
      Si apre la finestra Selezione template. Selezionare il template Macro e confermare con OK.
    4. Viene creata automaticamente una macro con il nome Makro0.
      Assegnare un numero diverso:
      • assegnare "57".
      • Oppure assegnare un numero diverso. In questo caso occorre modificare la variabile MakroNumberSpecialRun.
      (Non è possibile modificare la parte del nome macro.)
    5. Confermare con OK.
  2. Configurazione della macro di inizializzazione
    Se necessario modificare le seguenti variabili tramite la correzione delle variabili:
    • MakroNumberSpecialRun
    • TimeOutSpecialRun
MakroNumberSpecialRun
La variabile MakroNumberSpecialRun definisce quale, macro verrà chiamata dal controllo come macro di inizializzazione.
Default: 57
TimeOutSpecialRun
La macro di inizializzazione deve essere elaborata entro un tempo definito. In caso contrario, viene automaticamente deselezionata e il controllo emette il seguente messaggio:
Timeout durante l'esecuzione di macro di inizializzazione macro{0}. Deselezione di macro.
La variabile TimeOutSpecialRun definisce il tempo a disposizione.
Default: 1 000 ms
Consiglio: Impostare la variabile TimeOutSpecialRun a non oltre 2 000 ms.
Se si crea un altro modulo (sequenza, sottoprogramma o altra macro) mentre è in esecuzione una macro di inizializzazione, il controllo emette i seguenti messaggi di avvertimento. Alla luce del tempo di esecuzione relativamente breve, il numero di eventuali messaggi di avvertimento è limitato.
Il timeout non ha effetto se la macro è stata chiamata attraverso il modulo inline SPSMAKRO.
