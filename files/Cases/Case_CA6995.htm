KSS27002
Verifica ciclica per richiesta test freno non rispettata
Causa:Dopo una richiesta, non è stato svolto alcun test freni.
Descrizione
Il test freni è stato richiesto e il comando robot aveva emesso il messaggio Test freni necessario. In seguito, entro 2 h è necessario svolgere un test freni. Ciò non è avvenuto.
Rimedio:Eseguire un test freni.
Descrizione
L’esecuzione del test dei freni varia tra il KUKA System Software e il VW System Software.
Per informazioni più dettagliate sul test freni, consultare le istruzioni operative e di programmazione per gli integratori di sistema.
KUKA System Software
Il test freni è eseguito con il programma BrakeTestReq.SRC. L’avvio può avvenire nei seguenti modi:
  • Automatico
    Integrare il programma BrakeTestReq.SRC nell'applicativo, in modo che venga richiamato ciclicamente come sottoprogramma. Se viene richiesto un test freni, il programma lo rileva ed esegue subito il test richiesto.
  • Manualmente
    Avviare manualmente il programma BrakeTestReq.SRC.
VW System Software
Il test freni può essere realizzato nei seguenti modi:
  • Automatico
    A tale scopo è necessario creare la sequenza "Test freni" e collegarla all’applicazione in maniera tale che venga richiamata ciclicamente. Se viene richiesto un test freni, la sequenza lo rileva ed esegue subito il test richiesto.
  • Manualmente
    A tale scopo è necessario creare il sottoprogramma “Test freni” e avviarlo manualmente.
