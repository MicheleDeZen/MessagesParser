KSS01447
Punto non raggiungibile! Finecorsa software {Direzione di spostamento}{Numero di asse}
Causa:Fine-corsa software positivo non adatto per forza di saldatura elevate.
Descrizione
Con forze di saldatura elevate la pinza di saldatura si piega di più che con le forze di saldatura basse. Per questo motivo l’elettrodo mobile deve muoversi ulteriormente con forze di saldatura elevate negli elettroni fissi.
Internamente si calcolano punti eseguibili solo virtualmente. Essi rientrano tuttavia nella traiettoria. Maggiori sono la forza di saldatura e la piegatura della pinza, più profondamente sono calcolati anche i punti virtuali. Attraverso un crescente consumo delle cappe l’effetto è ulteriormente rinforzato.
Per controllare se il finecorsa software non è adatto per elevate forze di saldatura:
Istruzione verifica
  • Controllare se il punto di saldatura può essere saldato correttamente con forze di saldatura più basse.
Rimedio:Impostazione del finecorsa software positivo
Descrizione
Calcolare e impostare il valore corretto per il finecorsa software.
Presupposto
  • Gruppo utenti Esperto
  • La pinza è registrata.
  • È stato registrato il fattore di riduzione.
  • La pinza è calibrata.
  • Nuove cappe elettrodo
Procedimento
  1. Calcolare il valore per il finecorsa software positivo:
    Flessione max (mm) + Residuo Top cap + riserva (ca. 3 mm)
    I valori provengono dalle origini seguenti:
    • Flessione max (mm): Pagina di configurazione Parametro pinza
    • Residuo Top cap: Pagina di configurazione Configurazione
    • Riserva: il valore può essere all'occorrenza liberamente selezionato. Valore indicativo: 3 mm
  2. Riportare il valore calcolato sulla pagina di configurazione Parametro pinza in Pos. SoftEnd (mm) e salvare.
