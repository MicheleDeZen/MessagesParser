KSS03157
{Nome del modulo} : Nome più lungo di 24 caratteri
Causa:Nome modulo non ammissibile
Descrizione
Il nome del modulo non è ammissibile.
Nel nome del modulo sono ammessi solo i seguenti caratteri:
  • A-Z
  • a-z
  • 0-9
  • $
  • _
Inoltre, il nome del modulo non può contenere oltre 24 caratteri.
Rimedio:Rinominare il modulo
Descrizione
Rinominare il modulo.
Procedimento
  1. Nella struttura di directory marcare la cartella in cui si trova il file.
  2. Selezionare il file nella lista dei file.
  3. Selezionare Edit > Rinomina.
  4. Sovrascrivere il nome del file con uno nuovo e confermare con OK.
Nel gruppo utenti "Esperto" e nell'impostazione del filtro "Dettagli" sono riprodotti nel navigatore due file per modulo (file SRC e DAT). In tal caso bisogna rinominare entrambi i file!
