KSS03216
La via di accelerazione per CONST_VEL START in movimento in {Nome modulo@numero riga[:nome punto] del segmento limitante} è troppo corta. Invece di $VEL.CP = {Velocità programmata}m/s vengono raggiunti solo {Velocità raggiunta}m/s.
Causa:La corsa di accelerazione prima dell’area Velocità costante è troppo corta per raggiungere la velocità programmata.
Rimedio:Inizio della gamma corsa costante in direzione fine traiettoria spostato.
Descrizione
Se l’inizio della gamma corsa costante in direzione fine traiettoria è spostato, la corsa di accelerazione è più lunga.
Procedimento
  • Nell’istruzione CONST VEL START, programmare un valore superiore per PATH.
