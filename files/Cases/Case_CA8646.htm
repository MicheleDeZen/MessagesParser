KSS02162
Carattere inserito errato
Causa:Linebreak rimosso dal Programmcode
Descrizione
Nel Programmcode manca il carattere Linebreak (Carriage Return/Line Feed). Se si guarda il programma nell’editor KSS, esso non è riconoscibile perché il carattere non è visualizzato.
Per controllare se mancano caratteri nel Programmcode:
Requisito
  • Livello Windows
Istruzione verifica
  1. Aprire il file di programma in Notepad++.
  2. Nel menù Visualizzazione > Caratteri non stampabili attivare la voce Visualizza tutti i caratteri.
  3. Controllare se alla fine di ogni riga si visualizza il carattere <CR><LF>.
  4. Controllare se il file con la codifica dei caratteri ANSI è salvato.
Esempio di programma con Line Feed <LF> mancante
Se non è disponibile alcun Notepad++, per correggere il Programmcode, riprogrammare la riga alla quale compare l’errore. Non tagliare e reinserire la riga, perché così il Linebreak continuerebbe a mancare.
Rimedio:Correggere il programma
Procedimento
  • Correggere il programma
    Per maggiori informazioni, consultare la documentazione del rispettivo software.
