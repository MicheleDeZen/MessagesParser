KSS03191
Sensori attivi non possibili durante {Funzionalità incompatibile}
Causa:Utilizzo di un contesto RSI durante il movimento in una base accoppiata
Descrizione
Un contesto RSI è attivo. Il robot viene quindi mosso, per es. con RoboTeam, in una base accoppiata.
Esempio: Lo slave si muove nel sistema di coordinate di base del Master tramite LK_Base
Rimedio:Disattivare il contesto RSI prima del movimento nella base accoppiata
Descrizione
Prima della programmazione di comandi di movimento in una base accoppiata (per es. LK_Base) terminare il contesto RSI con RSI_OFF().
Procedimento
  1. Selezionare il programma nel navigatore e premere Apri. Il programma viene visualizzato nell'editor.
  2. Cercare ed elaborare il punto corrispondente nel programma.
  3. Chiudere il file e alla richiesta di conferma, se si intende salvare le modifiche, rispondere con Sì.
