KSS02125
Componente non appartiene al tipo
Causa:Si utilizza un componente non definito nel tipo di dati strutturali.
Rimedio:Non utilizzare i componenti o completarli nella definizione del tipo di dati strutturali.
Esempio di errore di programmazione
 1  DEF my_prog( )
 2  STRUC my_typ INT my_a, REAl my_b
 3  DECL my_typ my_var
 4  ...
 5  my_var.my_c = 5
 6  ...
Esempi di soluzione
Non utilizzare i componenti:
 1  DEF my_prog( )
 2  STRUC my_typ INT my_a, REAl my_b
 3  DECL my_typ my_var
 4  ...
 5  my_var.my_a = 5
 6  ...
Completare i componenti nel tipo di dati strutturali:
 1  DEF my_prog( )
 2  STRUC my_typ INT my_a, my_c, REAl my_b
 3  DECL my_typ my_var
 4  ...
 5  my_var.my_c = 5
 6  ...
