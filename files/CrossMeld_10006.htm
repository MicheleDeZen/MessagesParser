KSS10006
{Nome istanza} Errore apertura file {Nome file}
Effetto
Nessuna reazione di frenata
Nessun blocco di movimenti o comandi
Causa o cause possibili
1.Causa:La configurazione del file SVC è errata
Rimedio:Salvare il file di configurazione SVC tramite Config+ nella directory
2.Causa:Il file di configurazione SVC non è presente
Rimedio:Salvare il file di configurazione SVC tramite Config+ nella directory
3.Causa:La configurazione del file SVC è errata
Rimedio:Salvare il file di configurazione SVC tramite chiavetta USB nella directory
4.Causa:Il file di configurazione SVC non è presente
Rimedio:Salvare il file di configurazione SVC tramite chiavetta USB nella directory
